## 📝 CSV Upload Format Notes

### Column Mapping for Upload
```csv
facultyCollegeCode,facultyName,day,period,subjectCode,subjectName,classId,className,year,section,semester,roomNumber,labName,isLabSession,sessionType,academicYear,effectiveDate
```

### Important Field Values:
1. **facultyCollegeCode**: Use codes like CS10, NS80T01, NS50T08, etc.
2. **day**: Monday, Tuesday, Wednesday, Thursday, Friday, Saturday
3. **period**: 1, 2, 3, 4, 5, 6, 7
4. **roomNumber**: CR-15 (for theory classes)
5. **labName**: LAB-L1, LAB-MK, LAB-LAN (for lab sessions)
6. **isLabSession**: TRUE for labs, FALSE for theory
7. **sessionType**: theory, lab, tutorial
8. **academicYear**: 2025-2026
9. **effectiveDate**: 2026-01-05

### Lab Sessions Special Handling:
- Lab sessions span 2 periods → create 2 separate rows with same subject
- Set `isLabSession` = TRUE
- Set `sessionType` = "lab"
- Leave `roomNumber` empty
- Fill `labName` with lab code (LAB-L1, LAB-MK, LAB-LAN)
